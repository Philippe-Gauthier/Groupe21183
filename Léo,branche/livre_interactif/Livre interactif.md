# livre interactif
__________
## sommaire
Ce programme fonctionne avec une liste qui est allongé a chaque appui de boutons et remis pour le projet livre interactif du projet de programmation.
## but
il N'y a pas de but, je me suis simplement amuser a faire des sortie et entré dans le ficher.
## entrée
Le lecteur choisit entre 3 boutons qui vont décider ses actions
## sortie
le texte est sur la fenêtre donnée a l'utilisateur et pour voir si les boutons ont été appuyé un print a été fournie pour chaque appui de bouton
## Description de l'hierarchie des fichiers
tout fonctionne dans un seul fichier avec l'addition de customtkinter.
##
## Ce qui concerne customtkinter
vous pourrez l'installer en entrant
<pre><code>$ pip install customtkinter</code></pre>
pour le télécharger.
## structure
une structure en arbre qui permet a tout usager de faire des choix différents
