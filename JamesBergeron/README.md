# Mon projet

Mon projet est un livre intéractif où le joueur doit faire des choix entre certains objets pouvant l'aider dans ses futurs problèmes afin de déterminer son sort. 

# But
Atteindre une bonne fin de l'histoire avec un de ces objets si-dessous:
 - une plume de corbeau 
 - une boule de quilles

# Situation initiale

Le personnage arrive à l'intersection de deux sentiers, un à la droite, qui mène à une dense forêt et l'autre à la gauche, qui mène vers une grotte suivi d'un village inconnu.

# Entrées

La page présentée en ce moment indiquant les options que le personnage peut choisir afin de définir son futur sort. C'est les pages de mise en contexte qui mènent aux choix différents.

# Sorties
La page suivante selon le choix d'objet fait par le personnage. Jusqu'à une des six fins du livre. Les choix faitent par l'utilisateur pour choisir son sort.

# Les choix de réponses
Au début, il est important d'écrire "enter" pour commencer l'histoire intéractive.
Pendant l'histoire, Shade l'explorateur est confronté à de nombreux choix, dont ils sont identifiés par "a" et "b". Il y a une variété de possibilités au livre intéractif afin de montrer tous les chemins possibles.
