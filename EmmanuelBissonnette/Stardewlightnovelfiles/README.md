# Entree 
 lentree du joueur est un  numero de page qui sert a aller chercher un fichier texte

 # sortie

 la sortie est dans le terminal et il sagit dun texte avec une structure speciale voir structure texte si dessous (page) et parfois des immages faites en ascii art 
 # Strucutre

  1. Le joueur lit le texte et regarde les images affiches par le programe
  2. le joueur prend une desision 
  3. le joueur entre le numero de page correspondant a la reponse a la question quil veux donner 
  4. le jeux change de page 
  5. le jeux sauvegarde les items du joueur
  6. retour a letape 1
  7. jusqua se que le joueur recupere les items nessesaire
