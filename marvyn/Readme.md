# PROTOCOLE 0*6
## Description
Livre intéractif en python basé sur arbre décisionnel 40 choix.
Le jouer incarne un étudiaants  confronté a plusieurs décisions critiques qui influenent le déroulement de l'histoire .
   
Chaque choix modifie des variables permettant d'obtenir une fin différente. 


## Objectifs pédagogiques

Ce projet démontre l'utilisation de :
- Fonctions
- Modulùrités
- Validation des entrées utilisateur
- Variable décisionnel
- Arbre décisionnel
- Documentation du code
- Respect da norme PEP 8

## Fonctionnement du Jeu
Le jeu est divisé en plusieurs chapitres.

A chaque chapitre :

1. Une situation est présentée
2. Plusieurs choix sont proposés
3. Le joueur sélectionne une option
4. Des points sont ajoutés selon la décision

Deux variables principales sont utilisées :

- **Raison** = décisions logiques et prudentes
- **Ambition** = décisions rapides et stratégiques

À la fin du jeu, les points sont calculés pour déterminer la fin.



##  Les fins possibles

Selon les scores obtenus :

- LE PROTECTEUR = dominance de la raison
- LE CONTRÔLEUR = dominance de l’ambition
- L’ARCHITECTE = équilibre entre les deux
