# Projet 1: Livre interractif
## Concept
Mon projet est un livre intéractif de style «Goosebump» où le lecteur devra faire des choix qui influencera le sort du héro de l'histoire. Plusieurs fins différentes peuvent être atteignables.

## But
Réussir à arriver à une fin où le héro survit (ou simplement: ne pas mourrir)

## Structure du livre
### Entrées:
Les entrées sont les choix du lecteur. Le lecteur aura le choix entre 2 ou parfois 3 options pour continuer l'histoire.

### Sorties:
Les sorties sont le résultat du choix du lecteur, l'ammenant par la suite vers d'autres choix jusqu'à arriver à une des fins possibles.

### Structure:
Arbre, ce qui signifie que chacun des choix mènent vers un résultat différent.
